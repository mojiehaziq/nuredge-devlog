Novacor Web Package Deployment Guide
------------------------------------
1. Extract Novacor_Web_Package.zip.
2. Ensure all files remain in the same folder:
   - index.html
   - nuredge_privacy_policy.html
   - nuredge_terms_of_service.html
3. GitHub Pages Deployment:
   a. Create/open repository.
   b. Upload files to root or /docs.
   c. Commit & push changes.
   d. Go to Settings > Pages, set branch & folder, save.
   e. Access site from Pages URL.
